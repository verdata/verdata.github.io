Titanic Passenger Survival AnalysisÂ¶

IntroductionÂ¶

This study will analyse a dataset that contains demographic and passenger information from 891 passengers of the 2224 passengers and crew on board the Titanic and will investigate the following questions:

What factors made passengers more likely to survive? Did travelling with family have any impact on survival?

Data sources:

The data set used is available here and a description of the dataset can be found on the Kaggle website.

Data Wrangling and Data CleansingÂ¶

Missing Age DataÂ¶

Some age values are missing. The approach I decided to take was to fill the missing values with zero and ignore these passengers when investigating the age distribution and other statistics. (Filling with mean values would have distorted this data analysis.)

When creating the adult dataset I chose to include passengers with missing age data within it. This action increased the count of adults significantly (601 -> 778) and consequently should improve the statistical observations. There is a risk that some may have been children but the class and gender will be valid. I considered the benefit of having the additional rows in the adult anlysis outweighed this risk.

Other conversionsÂ¶

The set_passenger_class function is applied to the Pclass column to provide text values for class. The set_age_group function is applied to the Age column to provide text names for age ranges. These conversions make the following code intuitive and easier to understand.

Finally columns that are not used (name, ticket, fare and cabin) in the analysis are dropped from the dataframe, see data dictionary for details.

#utility functions to clean dataset

def set_passenger_class(pclass):
    """
    Return a text description for the numeric passenger class parameter
    unknown is default if 1|2|3 is not provided
    """
    pclass_name= "unknown"
    if pclass == 1 : pclass_name = "first"
    if pclass == 2 : pclass_name = "second"
    if pclass == 3 : pclass_name = "third" 
    return pclass_name  

def set_age_group(age):
    """
    Return an age group name derived from age parameter
    unknown is default if zero age provided
    """
    age = float(age)
    age_group_name = 'unknown'
    if  0 <  age <=  1 : age_group_name = 'infant'
    if  2 <= age <=  9 : age_group_name = 'child'
    if 10 <= age <= 17 : age_group_name = 'adolescent'
    if 18 <= age       : age_group_name = 'adult'
    return age_group_name    

def set_companions(value):
    """
    Return a text description for the numeric value parameter
    zero returns No otherwise Yes 
    """
    companions= "no"
    if value > 0 : companions = "yes"
    return companions




# Age columns has some NaN values, set these to zero 
passenger_data["Age"] = passenger_data["Age"].fillna(value=0 )

# Apply set_passenger_class function to Pclass column to provide text values for class

# Apply set_age_group function to Age_with_mean column to provide text names for age ranges

# Apply set_companions function to Siblings and Spouses column to provide Yes/No 
passenger_data["SibSp"] = passenger_data["SibSp"].apply(set_companions)

# Apply set_companions function to Parents and Children column to provide Yes/No 
passenger_data["Parch"] = passenger_data["Parch"].apply(set_companions)

# Drop columns that are not used in the analysis

# main function to analyse passenger dataframes
def calculate_survival_rate(df):
    """
    Expects a dataframe containing passenger information.
    Returns three values in a tuple:
    The total passenger count contained in the dataframe.
    The count of passengers that have the Survived column flag set to 1
    The survival rate obtained by dividing the previous values.
    """
    passenger_count = df.PassengerId.count()
    survived_count  = df[(df['Survived'] == 1)].PassengerId.count()
    return passenger_count, survived_count, survived_count / float(passenger_count)

    print("There are %d passengers in total; %d survived; survival rate: %f" % calculate_survival_rate(df))

def categorical_plot(data, key = '',key_name = '' ):
    """
    Plot number of passengers, by variable. 
   
    """
    # Create plotting figure
    plt.figure(figsize=(8,6))
    
    # Summarize dataframe to get counts in each group
    data['count'] = 1
    data = data.groupby(key, as_index = False).count()

    levels = data[key].unique()
    n_levels = len(levels)
    bar_width = 0.8

    for i in range(n_levels):
        trips_bar = plt.bar(i - bar_width/2, data.loc[i]['count'], width = bar_width)

    # add labels to ticks for each group of bars.
    plt.xticks(range(n_levels), levels)

    # Common attributes for plot formatting
    if key_name == ' ':
        key_name = ' '.join([x.capitalize() for x in key.split('_')])
    plt.xlabel(key_name)
    plt.ylabel("Number of Passengers")
    plt.title("Number of Passengers by {:s}".format(key_name))
    plt.show()

#Multi index plotting code adapted from
#https://stackoverflow.com/questions/31845258/pandas-multi-index-plotting
def add_line(ax, xpos, ypos):
    line = plt.Line2D([xpos, xpos], [ypos + .1, ypos],
                      transform=ax.transAxes, color='gray')
    line.set_clip_on(False)
    ax.add_line(line)

def label_len(my_index,level):
    labels = my_index.get_level_values(level)
    return [(k, sum(1 for i in g)) for k,g in groupby(labels)]

def label_group_bar_table(ax, df):
    ypos = -.1
    scale = 1./df.index.size
    for level in range(df.index.nlevels)[::-1]:
        pos = 0
        for label, rpos in label_len(df.index,level):
            lxpos = (pos + .5 * rpos)*scale
            ax.text(lxpos, ypos, label, ha='center', transform=ax.transAxes)
            add_line(ax, pos*scale, ypos)
            pos += rpos
        add_line(ax, pos*scale , ypos)
        ypos -= .1

def multi_index_plot(values,index,plot_title = ''):    
    data = pd.DataFrame(data={'val': values}, index=index)
    # unstack last level to plot two separate columns
    data = data.unstack(level=-1)
    ax = data['val'].plot(kind='bar')
    #Below 2 lines remove default labels
    ax.set_xticklabels('')
    ax.set_xlabel('')
    plt.title(plot_title)
    label_group_bar_table(ax, data)

ConclusionsÂ¶

What were the primary factors made passengers more likely to survive?Â¶

From our investigations certain combinations of factors: passenger class, gender and age had a clear impact.

All third class passengers have relatively poor survival rates regardless of age and gender, probably as a consequence of poor evacuation management decisions:

[...] an organized call for third-class passengers to proceed to the boat deck was not made until about fifty minutes after the collision. There was no realization that, given the deficiency of lifeboats, this was fatal to the chances of most third-class passengers. (Davenport-Hines, p.281)

Female passengers and children (both male and female) in first and second class have high survival rates. Correspondingly male passengers in first and second class have low survival rates (very low in the case of second class). By examining the standard deviations from mean for these adult survival rates we have concluded that these observations are statistically significant.

There are some secondary factors related to travelling with family, that influenced survival rates. My analysis showed that higher survival rates for the following groups are statistically significant:

  • first class men accompanied by siblings and spouses and third class women travelling alone
  • third class women without children and second class women with children

It could be argued that the results of the investigation simply confirm the cultural convention of 'women and children first' combined with poor evacuation management decisions made in a crisis situation.

Investigation Limitations and ConstraintsÂ¶

The analysis was performed using the entire sample of 891 passenger records and summarises their survival outcomes. We would expect to find similar survival outcomes for the rest of the passengers but because of the partitioning of the passenger population by age group, class and gender some groups have relatively low population numbers from which it may be unwise to extrapolate.

While this analysis is interesting in an historical and social perspective, it is not possible, due to the changes in maritime safety practice, to use this data to predict passenger survival for a modern day maritime disaster. It might be possible and interesting to compare analysis with contemporary events.

Historical Context

We are only able to consider the information available to us in the dataset, however, the time after the collision was very chaotic and it is likely that there were many extraneous factors in play that are not captured in the dataset.

Historical records provide us with some insight into what these might be.

We have assumed that all passenger classes had free access to the boat deck during the evacuation but in normal circumstances the passenger classes were segregated:

US immigration laws stipulated that passengers of different classes must be separated on liners by locked metal barriers to limit their supposed power to spread contagion, (Davenport-Hines,p.9)

During the evacuation it seems likely that the gates were opened but without a clear and well signed exit route, free access cannot be completely assumed:

The emergency door from third-class into second-class was opened early on. There was a route from there up the centre of the liner, past the second-class smoking room, straight to the boat deck. [...] some third-classers were baffled by the maze of corridors, and excluded by the locked gates that US immigration laws required between third-class decks and other parts of the ship, though these were unlocked by 12.30. (Davenport-Hines, p.281)

Another extraneous factor could be the reluctance of passengers and crew to use the lifeboats after the incident with P&O’s Oceana:

The recent capsizing in the English Channel of a lifeboat from P&O’s Oceana, which drowned nine people, discouraged passengers from being jerked down the tall precipitous side of a liner which was not expected to sink. They anticipated a delay in reaching New York, but nothing worse, and were reluctant to leave the warm comforts of the liner for the chill perils of a frail craft bobbing on the ocean. The P&O fatalities inhibited White Star’s officers, too, from filling their lifeboats. (Davenport-Hines, p.279)
